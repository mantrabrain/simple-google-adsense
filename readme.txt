== Description ==

Add google adsense code to your WordPress site. No need to configure too much, just put publisher ID from Settings->Google AdSense.


== Installation ==

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select simple-google-adsense.zip from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download yatra.zip
2. Extract the yatra directory to your computer
3. Upload the yatra directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard
